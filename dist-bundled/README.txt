MK-OrbitControl v1.0
====================
Menu bar monitor controller for Antelope Synergy Core interfaces.

Requirements:
- macOS 13 (Ventura) or later
- Antelope Launcher installed and running
- Antelope Synergy Core device

Tested Device:
- Antelope Orion Studio III (Synergy Core) ✅

Untested (should work but not verified):
- Discrete 4/8, Galaxy 32/64, Orion 32+, Zen Tour, Goliath
- Community testing and feedback welcome!

Installation:
1. Open the DMG
2. Drag MK-OrbitControl.app to Applications
3. Open Terminal and run: bash /Volumes/MK-OrbitControl/setup.sh
4. If macOS blocks the app: Right-click → Open → Open
5. The app icon appears in your menu bar

Features:
- Rotary volume knob with dB display (-∞ to 0 dB)
- DIM / MUTE / MONO buttons
- A/B monitor switching
- 4 preset slots (right-click to save, click to recall)
- Night mode (configurable volume cap)
- 12 themes + 8 fonts + 9 menu bar icons
- Global hotkeys (configurable per output)
- Peak meters (L/R with smooth decay)
- Volume HUD overlay on hotkey press

MK-OrbitControl v1.0
====================
Menu bar monitor controller for Antelope Synergy Core interfaces.

Requirements:
- macOS 13+
- Antelope Launcher installed and running
- Antelope Synergy Core device (Orion Studio III tested)
- Homebrew (for pyenv)

Setup:
1. Open Terminal
2. cd to this folder
3. Run: bash setup.sh
4. Double-click MK-OrbitControl.app

Features:
- Volume knob with dB display
- DIM / MUTE / MONO / Talkback
- A/B monitor switching
- 4 preset slots
- Night mode (volume cap)
- 12 themes + 8 fonts
- Global hotkeys
- Peak meters
- Volume HUD overlay
